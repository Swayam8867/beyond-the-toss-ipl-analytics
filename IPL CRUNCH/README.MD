# IPL CRUNCH '26 — IPL Data Analytics Project

## Overview

This project analyzes IPL ball-by-ball cricket data to uncover patterns influencing match outcomes, team performance, scoring behavior, and winning strategies.

The analysis was performed using Python, Pandas, Matplotlib, and Seaborn inside Jupyter Notebook.

---

## Objectives

- Analyze toss impact on match results
- Identify the most important match phase
- Discover top IPL teams
- Analyze top-performing batters and bowlers
- Study venue-based scoring patterns
- Generate meaningful cricket insights using data analytics

---

## Key Insights

- Toss advantage alone does not guarantee victory
- Death overs heavily influence match outcomes
- Consistent batting drives long-term success
- Certain venues strongly favor high-scoring matches
- Teams performing well under pressure dominate IPL standings
