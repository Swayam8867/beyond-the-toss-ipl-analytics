# Beyond The Toss: Decoding What Actually Wins IPL Matches

## Overview

This project explores IPL ball-by-ball match data to identify the real factors that influence match outcomes in T20 cricket.

Using Python and exploratory data analysis techniques, the project investigates:
- Toss impact
- Match phase dominance
- Batting consistency
- Bowling effectiveness
- Venue trends
- Death-over performance

The analysis converts raw IPL data into meaningful visual insights and strategic observations.

---

## Key Insight

One of the strongest findings from this project was that death-over performance influences IPL victories far more consistently than toss advantage.

Teams that perform strongly during the final overs tend to achieve better overall results across seasons.

## Major Findings

- Toss advantage alone does not guarantee victory
- Death overs strongly influence match outcomes
- Consistent batting drives long-term success
- Venue conditions impact scoring patterns
- Pressure handling is critical in T20 cricket
